[斐波那契数](https://leetcode-cn.com/problems/fibonacci-number/)

递归法

按照题意容易写出递归方法：

{% highlight python %}

class Solution:
    def fib1(self, N: int) -> int:
        res = [0, 1]
        if N <= 1:
            return res[N]
        else:
            return self.fib1(N-1) + self.fib1(N-2)

{% endhighlight %}
