[斐波那契数](https://leetcode-cn.com/problems/fibonacci-number/)

递归法

按照题意容易写出递归方法：

{% highlight python %}

class Solution:
    def fibonacci(self, n: int) -> int:
        if n == 0:
            return 0
        elif n == 1:
            return 1

{% endhighlight %}
