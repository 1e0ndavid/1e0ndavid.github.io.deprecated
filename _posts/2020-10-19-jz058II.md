这个比较简单了就不说了吧，这个算法的时间复杂度是O（N），空间复杂度是O（N）。

{% highlight python %}

class Solution:
    def reverseLeftWords(self, s: str, n: int) -> str:
        return s[n:] + s[:n]

{% endhighlight %}
