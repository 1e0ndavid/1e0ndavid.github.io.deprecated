这个算法的时间复杂度是O（N），空间复杂度是O（N）。

{% highlight python %}

class Solution:
    def add(self, a: int, b: int) -> int:
        x = 0xffffffff
        a, b = a & x, b & x
        while b != 0:
            a, b = (a ^ b), ((a & b) << 1) & x
        return a if a <= 0x7fffffff else ~(a ^ x)

{% endhighlight %}
