一个简单的dp题，空间可以优化到一维。这个算法的时间复杂度是O（MN），空间复杂度是O（N）。

python:

{% highlight python %}

class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [1]*n
        for i in range(1, m):
            for j in range(1, n):
                dp[j] += dp[j-1]
        return dp[-1]
